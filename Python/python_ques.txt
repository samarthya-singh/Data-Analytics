63)Write a Python function to check whether a number is perfect or not. 
Ans)
def is_perfect(number):
    if number <= 0:
        return False  # Only positive numbers can be perfect
    
    # Find sum of proper divisors
    sum_divisors = 0
    for i in range(1, number):
        if number % i == 0:
            sum_divisors += i
    
    # Check if sum of divisors equals the number
    return sum_divisors == number

# Example usage
num = 28
if is_perfect(num):
    print(f"{num} is a perfect number")
else:
    print(f"{num} is not a perfect number")


64)Write a Python function that checks whether a passed string is
palindrome or not 
Ans)
def is_palindrome(s):
    # Remove spaces and convert to lowercase for uniform comparison
    s = s.replace(" ", "").lower()
    
    # Check if the string is equal to its reverse
    return s == s[::-1]

# Example usage
string1 = "Radar"
string2 = "Hello"

print(f"{string1} -> {is_palindrome(string1)}")  # True
print(f"{string2} -> {is_palindrome(string2)}")  # False

83)Explain Exception handling? What is an Error in Python? 

Ans)
An error is an unexpected event that occurs during the execution of a program.

Errors stop the program from running unless handled.
#Exception handling

Exception Handling is the process of catching and managing errors (exceptions) so the program doesnâ€™t crash.

In Python, this is done using try, except, else, and finally blocks.

try:

Code inside this block is executed normally.

If an error occurs, Python looks for a matching except.

except:

Catches the specific exception or all exceptions if unspecified.

Allows you to handle the error gracefully.

else: (optional)

Runs only if no exception occurred.

finally: (optional)

Runs always, whether an exception occurred or not.

Useful for cleaning resources, like closing files.
